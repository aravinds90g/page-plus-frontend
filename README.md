# 🔥 PagePulse

A forbidden library floating in the void between stars — where ancient manuscripts meet quantum interfaces.

## ✦ Cosmic Design Philosophy

A dark cosmic book management system built with Next.js 14, React 18, TypeScript, and Tailwind CSS. Features particle constellations, volumetric light shafts, custom magnetic cursor, animated statistics, and a complete CRUD interface.

## 🎨 Chromatic Universe

- **Void Abyss** `#030307` — Event horizon black
- **Blood Moon** `#FF3D00` — Primary accent (dying star)
- **Solar Flare** `#FF6D2E` — Secondary fire
- **Nebula Ember** `#FF9B5C` — Distant warmth
- **Starlight** `#E8E3D9` — Primary text

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:3000](http://localhost:3000) to enter the archive.

## 📁 Structure

```
/app
  /library           — Main Archive (CRUD operations)
  /library/[id]      — Single Book Cosmic Reader
  /page.tsx           — Portal/Landing
  /layout.tsx         — Root with particles, cursor, orbs
/components
  /cosmic-ui          — All UI components
  /particles          — Canvas particle system
/hooks
  useBooks.ts         — CRUD + filter state management
/lib
  grimoireData.ts     — 15 seeded books with cosmic lore
  cosmicTypes.ts      — TypeScript interfaces
```

## 🌌 Features

- **Particle Constellation System** — Canvas-based dynamic particles that respond to mouse
- **Volumetric Light Shafts** — Animated light beams sweeping across background
- **Magnetic Cursor** — Custom cursor with orbital trail and hover effects
- **Animated Statistics** — Count-up numbers with orbital rings
- **Cosmic Cards** — Book cards with hover glow and orbiting particles
- **Wormhole Modal** — Add/Edit modal with portal ring animations
- **Supernova Delete** — Explosive delete confirmation
- **Solar Toast** — Notifications with solar flare animation
- **Full CRUD** — Create, Read, Update, Delete books
- **Search & Filter** — Real-time search and genre filtering
- **Responsive Design** — Works on all screen sizes

## 📚 Seeded Data

15 books with cosmic lore including Dune, The Midnight Library, Sapiens, Neuromancer, and more.
